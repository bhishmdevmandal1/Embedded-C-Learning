## Repository Overview
This repository contains Embedded C implementations developed across host and target environments to strengthen core programming concepts required for embedded firmware development.
### Host
Contains Embedded C programs executed on the host environment for learning and experimentation.
Covered areas include:
* Variables
* Data Types
* Operators
* Functions
* Function Return Values
* Arithmetic Operations
* Type Casting
* Global vs Local Variables
* Storage Classes
* Declaration vs Definition
* ASCII Concepts
* sizeof Operator
* Pointer Fundamentals

### Target
Contains STM32 target-side projects developed using STM32CubeIDE.
Current work includes:
* STM32 Hello World
* OpenOCD Semihosting
* Target-side experimentation and validation(STM32 Target Development Practice)

Hardware Platform:
* STM32F411RE Nucleo Board

### GDB
Contains C programming exercises and experiments developed using the OnlineGDB environment during Embedded C learning.
These programs are used to understand core C programming concepts before implementing them on embedded targets.
